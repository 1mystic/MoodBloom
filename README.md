MOODBLOOM
=========

Experimental workspace for liquid interfaces and agentic workflows.
Focus on organic interaction models, glassmorphism, and AI integration.

DIRECTORY STRUCTURE
-------------------

/flux
    Active development.
    Next.js (App Router), Bun, PostgreSQL, Prisma.
    Current implementation of the liquid interface.

/old-source
    Legacy prototypes and static HTML/CSS references.
    Contains original layout.html concept.

USAGE
-----

This repository is a monorepo-style collection.
Navigate to specific subdirectories for development.

    cd flux
    bun dev
